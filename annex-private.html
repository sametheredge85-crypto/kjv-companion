    How to Add a New Sermon

    1. Use the Preaching Assistant to verify and finalize your sermon.
    2. Ask it: “Generate the Annex entry for this sermon.”
    3. Save the generated HTML file into: /sermons/ (create this folder if it doesn’t exist).
    4. Add a link to the file in the list above.
    5. Commit and push to GitHub — it updates instantly.
